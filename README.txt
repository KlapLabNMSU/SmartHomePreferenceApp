When starting up the system
---------------------------

To start OPENHAB runtime:
 > E:\openhab
 > start.sh

To start webapp runtime: 
 > E:\xampp
 > xampp_start.exe

To start mongosh runtime:
 > E:\mongosh
 > mongosh.exe

To start SHDS runtime: (command prompt)
cd /d D:
 > E:\xampp\htdocs\SmartHomePreferenceApp
 > php SHDS.php



Using system
----------------
To access OPENHAB: (chrome)
 > http://localhost:8080/

To access webapp: (chrome)
 > http://localhost/smarthomepreferenceapp/home.php
